# reactJsPortfolio


## build process


```
npm install

npm install --global gulp
```

### New js build
`gulp js`

### CSS
`
gulp css
`
or 
`
gulp css  --prod
`


finally to run as local web server

`php server.php`

or

`open index.html`


## DEV-STEPS

While building use eslint to lint javascript

`gulp lint`
